Реализовать работу c API openweather.com (отправка запросов с параметрами, получение json ответа, разбор json на составляющие),добавить IO(ввод города, вывод в консоль/файл), реализовать обработку ошибок.

Ссылка на библиотеку Gson: https://github.com/google/gson/blob/master/UserGuide.md
Ссылка на документацию API: https://openweathermap.org/current


Текущее задание - добавить методов в класс JsonResponse.

1)Использование API openweather.com
2)Использование аргументов командной строки
3)поддержка командной строки (вида ProjectZero.exe -c Minsk -f 3 -> прогноз погоды в Минске на 3 дня),
